-HTML & CSS Quick-Reference Guide-

1. <HTML> Tag

The <HTML> tag identifies the document as an HTML document. It represents the root of an HTML document and acts as the container for all other HTML elements. All HTML documents start with <HTML> and end with </HTML>.

2. <HEAD> Tag

The <HEAD> tag defines the document header and does not affect the appearance of the document in the browser. It can include metadata like <title>, CSS links, and other information.

3. <TITLE> Tag

The <TITLE> tag defines the title of the document. It is placed between <HEAD> and </HEAD> tags. Each document can have only one title which should identify the document content. The title is not part of the document text and cannot contain links or markup.

4. <BODY> Tag

The <BODY> tag defines the main content of the HTML document, including text, images, lists, tables, and hyperlinks. You can set a background color or image using attributes:

  • <body bgcolor="#000000">
  • <body bgcolor="rgb(0,0,0)">
  • <body bgcolor="black">
  • <body background="http://profdevtrain.austincc.edu/html/graphics/clouds.gif">

5. Headings (<H1> – <H6>)

HTML has six heading levels. <H1> is the largest and boldest, <H6> the smallest. The ALIGN attribute can set alignment: LEFT, CENTER, or RIGHT.


<H1 ALIGN="CENTER">Level 1 Heading</H1>
<H2>Level 2 Heading</H2>
<H3 ALIGN="RIGHT">Level 3 Heading</H3>
<H4>Level 4 Heading</H4>
<H5 ALIGN="CENTER">Level 5 Heading</H5>
<H6>Level 6 Heading</H6>

6. Paragraph <P> Tag

The <P> tag creates paragraphs and spacing. Attributes:

  • align: left (default), center, right
  • dir: ltr (left-to-right), rtl (right-to-left)

<p align="center" dir="ltr">Centered text, left-to-right.</p>
<p align="right" dir="rtl">Right-aligned text, right-to-left.</p>

7. Line Break <BR> Tag

The <BR> tag ends the current line and moves to the next. Useful for multi-line content.

8. Horizontal Rule <HR> Tag

Creates a horizontal line separating content sections. Attributes:

  • align: left, center, right
  • size: height in pixels
  • width: width in px or %
  • color: line color
<hr align="center" size="5" width="50%" color="blue">

9. <FONT> Tag

Used to change font size, color, and style. Attributes:

  • size: 1–7
  • color: color name or RGB value
  • face: font type (e.g., Arial, Verdana)
<font size="3" color="blue" face="Arial">Sample Text</font>

Note: Modern practice is to use CSS instead of <FONT>.

10. Comments

HTML comments are ignored by the browser but visible in source code:

<!-- This is a comment -->

11. Superscript & Subscript (<SUP>, <SUB>)

<SUP> raises text above baseline: x2

<SUB> lowers text below baseline: H2O

12. Common CSS Properties

  • font-family
  • background-color
  • color
  • margin
  • padding
  • border-radius
  • box-shadow
  • text-align
  • max-width
